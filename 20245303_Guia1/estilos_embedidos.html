Estilos embedidos en las etiquetas HTML

Usa estilos CSS embedidos en las etiquetas HTML es el pero y el menos utilizado de los metodos para agregar reglas de estilo. No es nada recomendable por la dificultad que implica realiza un cambio a posteriori

Esta forma de incluir CSS directamente en los elementos HTML solamente se utiliza en determinadas situacionesen las que se debe incluir un estilo muy especifico para un solo elemento concreto

Para insertar un esulos CSS en un elemeto se utiliza la propiedad style de HTML, que en ningun momento se debe confundir con la etiqueta <style> que se intoduce en la cabecera del documento.
